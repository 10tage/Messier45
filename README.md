# Messier45
Objektdaten 
Katalog: Messier 45 
Entfernung: 444 
Lichtjahre Alter: ~100 Mio. Jahre 
Sterne: ~1000 (9 hell)

Teleskop Simulation mit Screenshot-Button, verschiedene Filter, Mouse-Zoom, Settings Panel
